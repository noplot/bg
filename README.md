# bg
bitgarden

## setup
- pull the repo (use SSH, we are professional here)
- node version: `v17.2.0`
- go version: `go1.17.3 darwin/amd64`
- `npm install`

## start
- `bg/front % npm run dev`
- `bg/back % go run server.go`
